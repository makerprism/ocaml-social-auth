# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-11-27

### Added
- Initial release of auth-provider-lwt
- Cohttp_lwt_unix HTTP client implementation
- Lwt promise-based wrappers for OAuth2 flows
- Error handling with Lwt.catch
- Type re-exports for convenience
- Complete OAuth2 flow support
- Token exchange and refresh
- User info retrieval
- Documentation and examples with Dream framework
