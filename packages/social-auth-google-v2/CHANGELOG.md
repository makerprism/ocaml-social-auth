# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-11-27

### Added
- Initial release of social-auth-google-v2
- Google OAuth 2.0 configuration with PKCE support
- OpenID Connect support
- User info parsing from Google userinfo v2 API
- Support for email, profile, and openid scopes
- Proper handling of `id` and `sub` fields
- Email verification status parsing
- Profile picture and locale extraction
- Scope validation utilities
- Documentation and examples
