# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-11-27

### Added
- Initial release of social-auth-github-v2
- GitHub OAuth 2.0 configuration with PKCE support
- User info parsing from GitHub API v3
- Support for read:user and user:email scopes
- Username and email extraction
- Avatar URL support
- Scope validation utilities
- Documentation and examples
